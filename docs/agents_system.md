# AGENT SYSTEM --- Plataforma Unificada

## Principios

-   Una responsabilidad por agente
-   No conflictos de código
-   Tareas pequeñas y verificables

## Agentes

### Orchestrator

Coordina tareas, divide trabajo y valida resultados.

### Frontend

Responsable de UI, componentes y experiencia.

### Backend

Responsable de base de datos, lógica y seguridad.

### QA

Valida flujos y detecta errores.

### Refactor (opcional)

Mejora código sin romper funcionalidad.

## Flujo

1.  Orchestrator recibe tarea
2.  Divide
3.  Asigna
4.  Ejecuta
5.  QA valida

## Reglas

-   No sobre-ingeniería
-   Validar cada paso
-   No modificar fuera del scope
